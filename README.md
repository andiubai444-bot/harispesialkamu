# Hari Spesial Kamu 💐

Sebuah website hadiah digital yang dibuat khusus untuk seseorang yang istimewa.

Website ini berisi rangkaian ucapan, kenangan, harapan, musik, animasi, dan pesan personal yang disusun menjadi pengalaman interaktif. Project ini dibuat sebagai bentuk apresiasi, rasa sayang, dan hadiah sederhana yang bisa dikenang kapan saja.

## ✨ Tentang Project

**Hari Spesial Kamu** adalah website personal berbasis web yang dirancang untuk memberikan pengalaman romantis dan emosional melalui tampilan visual yang lembut, animasi interaktif, serta alur cerita yang bertahap.

Project ini dibuat khusus untuk:

> Ameisha Nadilah  
> seseorang yang sangat berarti dan istimewa.

## 🌸 Fitur Utama

- Halaman pembuka interaktif
- Animasi bunga dan efek visual lembut
- Musik latar romantis
- Confetti celebration
- Section cerita dan pesan personal
- Counter waktu bersama
- Informasi jarak hubungan
- Countdown ulang tahun
- Pelukan virtual
- Tampilan responsif untuk desktop dan mobile
- Metadata Open Graph untuk preview link

## 🛠️ Teknologi yang Digunakan

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel
